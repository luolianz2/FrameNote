# 框记 FrameNote

跨端快速笔记系统。

## 解决的问题

上课时公式、图表等内容难以快速敲入 Markdown。手机端一键拍照采集，框选区域分离图片与文字，云端 OCR 识别后同步至电脑端，整理并导出为本地 `.md` 笔记。

## 技术栈

| 端     | 技术                   |
| ------ | ---------------------- |
| 手机端 | Android (Java)         |
| 云端   | Python (FastAPI)       |
| 电脑端 | Qt Widgets (C++)       |

## 数据流向

```
手机端                     云端                      电脑端
拍照采集 ──上传──→ 存储 + OCR ──同步──→ Markdown 编辑
                                            │
                                     导出 .md + 图片
```

## 开发计划

项目通过至少 6 次迭代逐步完善。
